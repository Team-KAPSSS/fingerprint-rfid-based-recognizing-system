Future Scope

In future, we would like to keep working on this project and make new additions to provide users with more advanced features and more detailed information. We have set our sights on the following additions in future-

  • We would like to add some high functional sensory on this system to make it more faster and more accurate.
  • We have a plan to make a database to store the fingerprint templates and the corresponding RFID UIDs to make it more smooth and dynamic for the users.
  • Beside this we also have a plan to change the microcontroller unit with an advance computing unit which supports the Wi-Fi and other subordinate connections.
  • We also have a plan to make an web page where the users can show their data easily.
